기능 정의
P 구분 기능명 상세 요구사항
P0 검색 회원 검색 닉네임 또는 휴대폰번호 기준으로 검색한다. 검색 버튼 클릭 또는 엔터로 실행된다. 검색어를 입력하지 않은 상태에서는 전체 유저를 노출한다.
P0 노출 회원 수 목록 상단에 전체 회원 수를 노출한다. 검색 결과가 있으면 결과 수로 변경된다.
P0 노출 회원 목록 노출 컬럼: No, 닉네임, 최근 접속 일시. 가입 일시 기준 내림차순(최근 가입자 상단) 정렬. 20명 단위 페이지네이션.
P0 이동 상세 진입 닉네임 클릭 시 해당 회원의 상세 페이지(ADM-MEM-002)로 이동한다.

· 검색 결과가 0건인 경우 → 목록 영역에 "검색 결과가 없어요." 문구 노출

기능 정의
P 구분 기능명 상세 요구사항
P0 이동 뒤로가기 뒤로가기 클릭 시 회원 목록(ADM-MEM-001)으로 이동한다. 기존 검색 상태는 유지되지 않는다.
P0 노출 기본 정보 노출 항목: 닉네임, 본명, 휴대폰번호, 가입일시.
P0 노출 모임 내역 탭 모임 내역 영역은 "가입된 모임" / "탈퇴한 모임" 탭으로 구성된다. 탭 레이블에 해당 건수를 표시한다. 기본 활성 탭은 "가입된 모임"이다.
P0 노출 가입된 모임 노출 항목: 모임명, 카테고리, 가입 일시, 역할(모임장/멤버). 가입 일시 기준 오름차순 정렬.
P0 노출 탈퇴한 모임 노출 항목: 모임명, 카테고리, 가입 일시, 탈퇴 일시. 탈퇴 뱃지로 표시. 탈퇴 일시 기준 내림차순 정렬.
예외 케이스
· 가입된 모임 탭: 가입 모임이 0개인 경우 → "가입된 모임이 없어요." 문구 노출
· 탈퇴한 모임 탭: 탈퇴 이력이 0개인 경우 → "탈퇴한 모임이 없어요." 문구 노출
· 존재하지 않는 회원 ID로 접근 시 → 회원 목록(ADM-MEM-001)으로 리다이렉트
